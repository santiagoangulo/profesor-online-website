# Project: Maths Teacher Website

## Overview

This project is a marketing website for a mathematics teacher. The site promotes tutoring services and allows booking via Calendly.

## Main Features

- Home page with hero, Mathematics focus, teaching method, and testimonials preview
- Services page with Mathematics details and pricing tiers
- Booking page with Calendly integration
- Testimonials page with student success stories
- Educational resources section (material-didactico)
- Bilingual support: Spanish (primary) and English
- Dark/light theme with system preference detection
- Mobile-first responsive design
- SEO-friendly with meta tags and sitemap

## User Behavior

- Students navigate the website to learn about the teacher and services
- Students book a free diagnostic class via Calendly
- Students access free educational resources
- The website should be easy to navigate and accessible to all users

## Tech Stack

- 11ty (Eleventy) - Static site generator
- Nunjucks templates (.njk)
- Plain CSS (base.css, style.css, inner-pages.css, home.css)
- Vanilla JavaScript (app.js)
- Netlify for deployment

## Design Goals

- Clean and modern marketing website
- Hero-driven layout with clear CTAs
- Mathematics cards, pricing grids, testimonials masonry
- Google Meet branding throughout
- Mobile-first responsive design

## Scope Limitations

- No authentication
- No backend/database
- No blog section
- No contact form (booking via Calendly only)
- No user accounts or dashboards
